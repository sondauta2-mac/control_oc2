# 🤖 Sistema de Seguimiento Automatizado de Órdenes de Compra

Prueba de Concepto (POC) validada de forma exitosa para la automatización de consultas de Órdenes de Compra (OC), eliminando la carga de correos y llamados telefónicos.

## 🚀 Componentes del Repositorio
*   `app.py`: Código fuente de la interfaz del Chatbot Web (Streamlit).
*   `cargar_datos.py`: Script de simulación que inyecta 50 registros aleatorios en la base de datos local.
*   `compras.xlsx`: Base de datos plana estructurada para la simulación de la POC.
